Brushed-aluminium surface — the brand's core material. Use as a panel/body behind labels, or as a nameplate.

```jsx
<MetalPlate style={{ width: 240, height: 120 }} />
<MetalPlate tone="dark" radius={8} style={{ padding: 24 }}>…</MetalPlate>
```

Tones: `aluminium` (default brushed silver), `dark` (matte black cap/lid). `sheen` toggles the top highlight.

Tracked-caps action button in the VIS MAJOR house style — monochrome ink/paper, use for any primary or secondary CTA.

```jsx
<Button variant="primary" size="lg">Shop GY-NO!</Button>
<Button variant="secondary">Learn more</Button>
```

Variants: `primary` (ink fill), `secondary` (outline), `ghost` (text only). Sizes `sm | md | lg`. Pass `full` to stretch, `as="a"` + `href` for links.
